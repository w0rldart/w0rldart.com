    • Configuring UTF8 Collation for MySQL or Mariadb on Docker

    May 22, 2018

    Configuring UTF8 character-set/collation for your MySQL or MariaDB service on a Docker container can be achieved by passing the following parameters in the command section

    • --character-set-server=utf8
    • --collation-server=utf8_unicode_ci
    • --init-connect='SET NAMES UTF8;'
    • --innodb-flush-log-at-trx-commit=0
